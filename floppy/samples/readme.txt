Mitsumi samples have been edited from Shiru's recordings.

Originals can be found here:
http://shiru.untergrund.net/files/flopster.zip

Original Mitsumi samples can edited and used separately from the Flopster VSTi under the
Creative Commons Attribution (CC-BY) license terms.

Rest of the samples created for 86box are released under the same license.

The samples are 48kHz 16bit 1 channel PCM WAVE samples. 86box code does not make any resampling
and assumes that the host audio system is using 48kHz 16bit output.
